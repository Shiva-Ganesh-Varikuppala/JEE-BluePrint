# JEE Blueprint

A dark-only personal JEE study planner.

## Run locally

```bash
# API
cd backend
python3 -m pip install --user -r requirements.txt
python3 app.py

# Frontend (a second terminal, from project root)
npm install
npm run dev
```

The frontend runs on port 5173 and the Flask API runs on port 5050.

See [docs/Roadmap.md](docs/Roadmap.md) for the feature plan.
