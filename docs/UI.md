# UI direction

- Dark-only premium interface with muted charcoal surfaces, violet accents, rounded cards, and strong reading contrast.
- Desktop layout uses a persistent sidebar; mobile layout uses a slide-in sidebar.
- The syllabus is reached through the persistent **View full syllabus** action and supports subject filters and search.

## Core pages

| Page | Purpose |
|---|---|
| Auth | Personal account registration and sign-in |
| Dashboard | Daily overview, goals, progress, and shortcuts |
| Todo | Create and manage the day's study commitments |
| Syllabus | Track all 92 chapters by subject and school year |
| Planner | Organize study blocks across the week |
| Revision | See what is due using spaced repetition |
